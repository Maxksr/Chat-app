# Real-Time Java Chat Application

A multi-client real-time chat application. The project uses a client-server architecture where multiple users can connect to a server and exchange messages instantly.

The server handles multiple clients using multithreading, while socket communication enables fast and reliable message exchange between connected users. 

### Features

* Real-time messaging between multiple clients
* Multi-threaded server architecture
* TCP socket-based communication
* User connection and disconnection notifications
* Buffered input/output stream handling

### Technologies

* Java
* TCP Sockets
* Multithreading
* Object-Oriented Programming (OOP)
